# 🌿 Cotton Plant Disease Prediction Using Hybrid Deep Learning

## 📌 Overview
This project focuses on the automated detection and classification of cotton plant diseases using a hybrid deep learning architecture combining DenseNet121 and ResNet50. Early disease detection helps farmers reduce crop loss and improve agricultural productivity.

## 🚀 Features
- Automated cotton disease classification
- Hybrid CNN architecture (DenseNet121 + ResNet50)
- Image preprocessing and augmentation
- Disease prediction with high accuracy
- Performance evaluation using confusion matrix and classification metrics

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

## 📂 Dataset
The dataset contains cotton leaf images belonging to:
- Healthy Cotton Leaf
- Bacterial Blight
- Curl Virus Disease
- Fusarium Wilt
- Other Common Cotton Diseases

Images were manually annotated and preprocessed before training.

## 🧠 Model Architecture
- DenseNet121 for deep feature extraction
- ResNet50 for residual learning
- Feature fusion for improved classification performance
- Softmax output layer

## 📊 Results
- Accuracy: 96%
- Precision: High
- Recall: High
- F1 Score: High

Evaluation includes:
- Accuracy Curve
- Loss Curve
- Confusion Matrix
- Classification Report

## 📸 Output
The model predicts cotton leaf diseases from uploaded images and provides disease classification results.
